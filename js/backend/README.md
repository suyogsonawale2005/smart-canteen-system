# 🍽️ Smart Canteen Management System

## 📌 Project Overview
A web-based canteen management system that allows students to order food online and admins to manage menu, orders, and reports.

---

## 🚀 Features

### 👨‍🎓 Student
- Signup & Login
- Browse Menu (Veg / Non-Veg / Categories)
- Add to Cart & Place Order

### 🛠️ Admin
- Secure Login
- Dashboard with:
  - Total Revenue
  - Total Orders
  - Daily / Weekly / Monthly Reports
- Menu Management

---

## 🧠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Database: MySQL (XAMPP)

---

## 📊 Database Tables
- customers
- admin
- menu
- orders
- order_items
